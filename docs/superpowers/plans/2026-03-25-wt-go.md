# wt go Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `wt go <branch-or-slug>` command that navigates to an existing worktree by printing its path to stdout for the shell wrapper to `cd` into.

**Architecture:** `cmd_go()` iterates all registered projects (same registry loop as `cmd_list`), matches the arg against `branch` or `slug` fields in `.wt-meta`, and prints the single matching worktree path to stdout. Shell wrapper already handles the `cd`; only the condition list needs `"go"` added. The command is pure lookup — no git operations.

**Tech Stack:** Bash 4+, `bin/wt` (single-file CLI), `bin/wt-test` (integration tests)

- [ ] **Step 1: Add `cmd_go()` function**

Insert the following immediately after the closing `}` of `cmd_list` (after line 1077):

```bash
# wt go <branch-or-slug> — navigate to an existing worktree
cmd_go() {
  local arg="${1:-}"
  [[ -z "$arg" ]] && error "Usage: wt go <branch-or-slug>"

  local branch_matches=()
  local slug_matches=()

  while IFS= read -r meta_file; do
    [[ -f "$meta_file" ]] || continue
    local branch slug wt_dir
    branch=$(read_meta "$meta_file" "branch")
    slug=$(read_meta "$meta_file" "slug")
    wt_dir=$(dirname "$meta_file")
    [[ -z "$branch" ]] && continue
    [[ "$branch" == "$arg" ]] && branch_matches+=("$wt_dir")
    [[ "$slug"   == "$arg" ]] && slug_matches+=("$wt_dir")
  done < <(
    for project_root in "${REGISTERED_PROJECTS[@]+"${REGISTERED_PROJECTS[@]}"}"; do
      [[ -d "$project_root/${WORKTREES_DIR}" ]] || continue
      for meta_file in "$project_root/${WORKTREES_DIR}"/*/"$META_FILE"; do
        [[ -f "$meta_file" ]] || continue
        echo "$meta_file"
      done
    done
  )

  # Branch match takes priority over slug match
  local matches=()
  if (( ${#branch_matches[@]} > 0 )); then
    matches=("${branch_matches[@]}")
  else
    matches=("${slug_matches[@]}")
  fi

  case "${#matches[@]}" in
    0) error "No worktree found for: '$arg'" ;;
    1) echo "${matches[0]}" ;;
    *)
      warn "Ambiguous: multiple worktrees match '$arg':"
      for m in "${matches[@]}"; do
        echo "  $m" >&2
      done
      error "Run 'wt list' to see all worktrees"
      ;;
  esac
}
```
